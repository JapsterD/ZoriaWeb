# ZoriaWeb

A landing page and marketing website for the Zoria project (ai16zoria.xyz), built on **BNB Smart Chain (BSC)** ecosystem branding and compatible with static hosting.

## Technology Stack

- **Frontend**: Static HTML/CSS/JS
- **Tools**: Webflow export, jQuery
- **Hosting**: Any static file server (Nginx, Apache, Vercel, GitHub Pages)

## Supported Networks

- Marketing site — references BNB Smart Chain and $ZORIA token ecosystem

## Contract Addresses

| Network | Token Contract ($ZORIA) |
|---------|-------------------------|
| BNB Mainnet | `0x0B71296D09B5aa459c6c79A425e41Aa9179D7777` |

## Features

- **Responsive design** — Mobile and desktop layouts
- **Zoria branding** — Logos, icons, ecosystem visuals
- **Gem Hunter page** — Dedicated subpage
- **Lightweight** — Static assets, fast load
- **BSC ecosystem** — Links and references to $ZORIA on BNB Chain

## Structure

- `index.html` — Main landing page
- `gemhunter.html` — Gem Hunter page
- `main.css` — Styles
- `icon/` — Icons and logos

## Usage

```bash
# Serve locally
npx serve .
# or
python -m http.server 8000
```

Open `index.html` in a browser or deploy to any static host.
