# ZoriaWeb

Landing page and marketing website for the Zoria project (ai16zoria.xyz).

## Structure

- `index.html` — Main landing page
- `gemhunter.html` — Gem Hunter page
- `main.css` — Styles
- `icon/` — Icons and logos

## Usage

Open `index.html` in a browser or serve the folder with any static file server:

```bash
npx serve .
# or
python -m http.server 8000
```

## Tech

Static HTML/CSS/JS. Built with Webflow export.
